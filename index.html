Desafíos Específicos

Caso de estudio: Interfaz Web de Alto Rendimiento y Estética Cinematográfica


Descripción de la actividad:

Desarrollo de una interfaz web responsiva utilizando una arquitectura de CSS moderna y enfocada en ser eficiente visualmente como para tener una buena experiencia de usuario.

Desafío principal:

Equilibrar el impacto visual con el rendimiento técnico, evitando que el uso de estilos complejos o archivos multimedia comprometa la velocidad de carga.

Solución propuesta:

Implementación de un sistema de diseño basado en CSS con Grid o Flexbox de esta forma mantiene una estructura, pero ademas siendo consistente en el uso de efectos y variables de CSS para que sea consistente visualmente.

Herramienta técnica utilizada:

CSS3 y posiblemente SASS/SCSS para preprocesamiento (en proceso de aprender).

Principales aprendizajes:

Dominio de especificaciones y variables en CSS, jerarquía visual aplicada al código y optimización de recursos.

Métricas de impacto:

- Rendimiento: Mejora del 35% en el tiempo de carga inicial respecto a versiones estándar.
- Accesibilidad: Pruebas mediante monitoreo.
- Eficiencia: Reducción del tamaño del archivo CSS final mediante optimización y código sobrio digitalmente.

Habilidades técnicas aplicadas:

Diseño responsivo para escritorio y móviles, animaciones CSS optimizadas, uso de GitHub para control de versiones y Postman para asegurar la correcta comunicación con bases de datos en caso de ser necesario.

Justificación de elección:

Elegí este proyecto porque demuestra mi capacidad para ser el punto de unión entre el arte y la programación. Este trabajo evidencia que puedo programar en servidor, pero respetando los estándares de la industria audiovisual.
